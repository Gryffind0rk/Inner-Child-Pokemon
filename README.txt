INNER CHILD CARDS INC. v4 WEB APP

HOW TO USE LOCALLY:
1. Unzip this folder.
2. Open index.html in Chrome, Edge, or Safari.
3. Add cards/sealed products manually.
4. Use Export Backup regularly to save your data.

HOW TO MAKE IT OPEN FROM ANYWHERE:
Upload these files to a static host such as Netlify, GitHub Pages, Vercel, or your own web host.
Then visit the hosted URL from your laptop or phone.

IMPORTANT:
This version stores data in your browser local storage. Export backups before clearing browser data or switching devices.
Cloud login/sync can be added in a future version using Firebase/Supabase.
