How to Use Release 1.0

1. Use Cards for what you own: cost, grade, quantity, notes.

2. Use Market Data for current prices. This fixes the issue where market value looked the same as cost.

3. Use Export Backup regularly. The app stores data in your browser, so backup/import protects your collection.

4. Upload this folder to GitHub Pages whenever you want a fresh hosted version. Your browser data is local to each device unless you import the backup there.
