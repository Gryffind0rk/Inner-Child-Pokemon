# Inner Child Cards Inc. — Release 1.0

This is a single-file browser app for tracking Pokémon cards, sealed products, watchlist items, goals, and manual market data.

## How to open locally
Double-click `index.html`.

## How to upload to GitHub Pages
Upload `index.html` to the root of your existing GitHub repository, replacing the old one. Then wait for Pages to rebuild.

## Important
This app uses browser localStorage. Use **Export Backup** often. If you open the website from a new phone/laptop, import your backup JSON to bring your data over.

## Market data model
Market data is independent from cost basis. Update Raw, PSA 9, and PSA 10 values in the Market Data tab.
