INNER CHILD CARDS INC. - DEPLOY READY WEB APP

WHAT THIS IS
This folder is ready to upload to GitHub Pages or Netlify.
The main file is index.html. Do not upload the ZIP itself; upload the extracted files.

QUICK GITHUB PAGES SETUP
1. Go to github.com and sign in.
2. Click + > New repository.
3. Name it: inner-child-cards
4. Set it to Public.
5. Click Create repository.
6. Click uploading an existing file.
7. Drag ALL extracted files into GitHub: index.html, manifest.json, sw.js, netlify.toml, README files.
8. Click Commit changes.
9. Go to Settings > Pages.
10. Under Source, select Deploy from a branch.
11. Branch: main. Folder: /root.
12. Click Save.
13. Your app will appear at: https://YOUR-GITHUB-USERNAME.github.io/inner-child-cards/

NETLIFY SETUP
1. Go to app.netlify.com/drop.
2. Drag this extracted folder onto the page.
3. Netlify gives you a live URL immediately.

HOW DATA IS SAVED
This v4 app stores data in your browser's local storage. Use Export Backup inside the app after changes.
If you use it on multiple devices, import/export the backup file between devices.

IMPORTANT
This is a local/static web app. It does not currently sync automatically between devices. A future version can add login/cloud sync.
